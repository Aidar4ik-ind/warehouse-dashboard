# Система оптимизации размещения товаров в сортировочном центре

Разработано в рамках ВКР. Реализует математическую модель линейного программирования (PuLP) и веб-интерфейс.

## Локальный запуск

1. Установите Python 3.9+.
2. Склонируйте репозиторий и перейдите в папку `backend`.
3. Установите зависимости:  
   `pip install -r requirements.txt`
4. Запустите сервер:  
   `python app.py`
5. Откройте в браузере адрес `http://127.0.0.1:5000`

## Развёртывание на бесплатном хостинге (Render)

1. Залейте код на GitHub.
2. На [Render.com](https://render.com) создайте новый Web Service, подключите репозиторий.
3. Укажите:
   - **Environment**: Python
   - **Build Command**: `pip install -r backend/requirements.txt`
   - **Start Command**: `python backend/app.py`
4. Сервис получит публичный URL – это и будет ваш сайт.

## Возможности
- Ввод объёмов товаров и вместимостей зон.
- Решение задачи оптимизации (симплекс-метод).
- Отображение оптимального распределения, времени, процента улучшения.
- Визуальные шкалы заполненности зон.
- Стиль соответствует требованиям ГОСТ (белый фон, строгая табличная форма).

## Структура
- `backend/` – Flask-сервер и модуль оптимизации.
- `static/` – CSS.
- `templates/` – HTML-интерфейс.
